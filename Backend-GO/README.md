# Backend Summarizer

## Struktur Project

- **cmd/main.go**: Entry point aplikasi
- **controller/**: Handler HTTP
- **service/**: Logika pemrosesan ringkasan
- **database/**: Koneksi & operasi DB
- **model/**: Definisi model data
- **router/**: Setup routing

## Cara Menjalankan
   go mod tidy
   go run cmd/main.go
